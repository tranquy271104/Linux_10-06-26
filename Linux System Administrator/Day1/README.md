## 📌 Bài 01: Giới thiệu về khóa học Linux System Administrator Exam 101 & 102

## 🗂️ Nội Dung Tổng Quan Khóa Học

Khóa học trang bị toàn diện kiến thức cốt lõi về hệ điều hành Linux từ cơ bản đến nâng cao:
* **Hệ thống & Phần cứng:** Kiến trúc hệ thống, Linux File System, cách Linux tương tác với thiết bị phần cứng.
* **Quản trị cốt lõi:** Quản lý User/Group, cấu hình hệ thống Log (System Logging), bảo trì và bảo mật hệ thống Linux Server.
* **Tự động hóa & Môi trường Shell:** Làm chủ các lệnh GNU/Unix, viết kịch bản tự động hóa bằng Bash Script.
* **Mạng & Dịch vụ nâng cao:** Cấu hình Linux Networking, cấu hình Web Server, Database Server và ảo hóa với Docker Container.

---

## 🎯 Đối Tượng Học Viên & Yêu Cầu Đầu Vào

### Đối tượng phù hợp
* Bất kỳ ai muốn bắt đầu học về hệ điều hành Linux.
* Quản trị viên hệ thống, kỹ sư mạng, lập trình viên, sinh viên ngành CNTT/Điện tử viễn thông.
* Người chuẩn bị lộ trình học DevOps và Cloud (AWS, Azure, Google Cloud).
* Học viên có mục tiêu thi lấy chứng chỉ quốc tế LPIC-1.

### Yêu cầu kiến thức nền tảng
* Có hiểu biết căn bản về hệ điều hành nói chung (Windows/macOS...).
* Khả năng đọc hiểu tiếng Anh cơ bản (thuật ngữ chuyên ngành).
* Tính kiên trì, nỗ lực thực hành Lab liên tục.

## 🏅 Thông Tin Chứng Chỉ LPIC-1

* **Giá trị hiệu lực:** **5 năm**. Tổ chức LPI khuyến nghị nên gia hạn sau **2 - 3 năm** (bằng cách thi lại hoặc thi lên LPIC-2 trước khi LPIC-1 hết hạn) để chứng chỉ luôn duy trì trạng thái *Active*.
* **Điều kiện cấp chứng chỉ:** Phải thi đỗ cả **2 bài thi** bắt buộc là **Exam 101** và **Exam 102**.

### 📊 Cấu Trúc Đề Thi (Áp dụng cho cả 101 và 102)
* **Thời gian:** 90 phút / bài thi.
* **Số lượng:** 60 câu hỏi.
* **Hình thức:** Trắc nghiệm khách quan (Multiple choice) và Điền vào chỗ trống (Fill-in-the-blank).

## 📝 Lộ Trình Chi Tiết 2 Bài Thi

### 1. Đề thi LPIC-1 (Exam 101)
Bao gồm 4 chủ đề chính:
* **Chủ đề 1:** Kiến trúc hệ thống (*System Architecture*).
* **Chủ đề 2:** Cài đặt và quản lý ứng dụng phần mềm (*Linux Package Management*).
* **Chủ đề 3:** Các câu lệnh GNU và Unix (*GNU and Unix Commands*).
* **Chủ đề 4:** Thiết bị lưu trữ và tiêu chuẩn phân cấp hệ thống file (*Devices, Linux Filesystems, Filesystem Hierarchy Standard*).

### 2. Đề thi LPIC-1 (Exam 102)
Bao gồm 6 chủ đề chính:
* **Chủ đề 1:** Môi trường Shell và viết kịch bản Shell (*Shells and Shell Scripting*).
* **Chủ đề 2:** Giao diện người dùng và đồ họa Desktop (*Interfaces and Desktops*).
* **Chủ đề 3:** Các tác vụ quản trị hệ thống cốt lõi (*Administrative Tasks*).
* **Chủ đề 4:** Dịch vụ hệ thống cơ bản (*Essential System Services*).
* **Chủ đề 5:** Kiến thức mạng căn bản (*Networking Fundamentals*).
* **Chủ đề 6:** Bảo mật hệ thống (*Security*).

> 🌐 **Tài liệu tham khảo chính thức:** Tra cứu chi tiết mục tiêu phòng thi tại trang chủ [LPI Certification Obijectives](https://www.lpi.org).
